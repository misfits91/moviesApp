# Movies App

### Descripción
Aplicación de prueba de iOS. Muestra un listado de peliculas de la API de [The Movie DB](https://www.themoviedb.org/) con sus detalles.

### Instrucciones
- Clonar repositorio `https://github.com/misfits91/moviesApp.git`
- `npm install` ó `yarn`
- `cd ios && pod install`
- `cd .. && npm run ios`
